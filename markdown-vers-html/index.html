Markdown

Qu'est le MarkDown

Markdown est un langage de balisage qui a pour but est d'offrir une syntaxe facile à lire et à écrire.

C’est-à-dire qu’un document formaté selon Markdown devrait pouvoir être publié comme tel, en texte, sans donner l’impression qu’il a été marqué par des balises ou des instructions de formatage Markdown est donc un moyen d’ajouter du formatage à un texte (à la manière du HTML) mais sans utiliser de menus ou de clics de souris: juste en utilisant une ponctuation spéciale !

La syntaxe

Titre

# This is an h1 tag

## This is an h2 tag

### This is an h3 tag

###### This is an h6 tag

Emphasis

*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

_You can combine them_

Lists

Unordered

  • Item 1
  • Item 2
    • Item 2a
    • Item 2b

Ordered

  1. Item 1
  2. Item 2
  3. Item 3
    1. Item 3a
    2. Item 3b

Images

![GitHub Logo](/images/logo.png)
Format: ![Alt Text](url)

Links

Link to GitHub !

Tables

First Header Second Header
Content from cell 1 Content from cell 2
Content in the first column Content in the second column
